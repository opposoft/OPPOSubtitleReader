## 🏗️ О проекте

**OPPO Subtitle Reader** — это проект для тех, кто хочет смотреть зарубежный контент, но не хочет или не может постоянно отвлекаться на чтение текста на экране.

Здесь автоматические или оригинальные субтитры YouTube превращаются в закадровую озвучку, чтобы можно было:
- слушать видео фоном (как подкаст);
- узнавать новое без знания языка;
- снимать нагрузку с глаз при просмотре.

Скрипт сделан максимально просто:
**без шума, без перегруза, без лишнего интерфейса**.

---

## ⚡ Особенности

- 🗣️ озвучка любых субтитров на лету
- 🧹 умная очистка текста (игнорирует теги `[смех]`, `(музыка)` и знаки `>>`)
- 🖱️ плавающая и перетаскиваемая панель управления
- 🎚 выбор языка и точная настройка скорости чтения
- 💾 сохранение настроек и позиции окна в `localStorage`
- 🌐 работа через встроенный Web Speech API
- ⏸️ автоматическая пауза при остановке видео
- 📱 не перекрывает элементы управления плеером

---

## 🧠 Идея

Этот проект создан как **инструмент для преодоления языкового барьера**.

Цель — дать комфортный доступ к мировому контенту, чтобы пользователь мог:
- потреблять информацию удобным способом;
- не привязываться к визуальному контакту с экраном;
- экономить время, занимаясь своими делами под звучащее видео.

**OPPO** не переводит сам. **OPPO** читает для вас!

---

## 🌍 Технологии

Все голоса для озвучки берутся из **встроенных возможностей вашей ОС и браузера**.

Проект:
- не использует сторонние нейросети и API;
- не отправляет текст на чужие серверы (всё работает локально);
- не требует подписки;
- не внедряет рекламу.

Так как это пользовательский скрипт (Userscript), для его работы требуется расширение-менеджер.
